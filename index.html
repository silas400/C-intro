Creating Variables in C!

int age;

float height;

In C, variables are created by specifying the type followed by the variable name. Here, int is used for integer variables, and float is used for floating-point variables.

Initializing Variables!

int age = 25;

float height = 5.9;

Variables can be initialized with values at the time of creation. This means assigning a value to the variable using the equal sign =.

CHALLENGE!

Create a variable to store the number of students in a class and initialize it with the value 30. Also, create a variable to store the average grade as a float and initialize it with the value 85.5.

Updating Variables!

age = 26;

height = 6.0;

Variables can be updated with new values after they have been initialized. This is done by reassigning a new value to the variable.

Using Variables in Expressions!

int totalYears = age + 5;

float newHeight = height + 0.1;

Variables can be used in expressions and arithmetic operations. Here, we are adding values to the existing variables age and height.

Combining Variables (Step by Step)

int width = 10;

Step 1: Create an integer variable named width and initialize it with the value 10.

int length = 20;

Step 2: Create another integer variable named length and initialize it with the value 20.

int area = width * length;

Step 3: Create an integer variable named area and initialize it with the product of width and length.
